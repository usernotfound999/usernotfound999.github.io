自定义开机第一屏教程

日期:2025年3月7日

此操作适用于1+，黑厂，真我
1.准备工作
(1)提取splash
用你喜欢的方式提取splash，软件提取，全量包提取......用你喜欢的方法，只要提取的所在插槽的splash就行，放到你能找到到的地方，划重点这里待会要考
(2)安装termux
在手机上安装termux，去Github下载个就行，访问Github我不负责
①在termux安装软件包
在termux里输入以下命令
pkg install wget imagemagick
然后回车
②下载opsplash
用任意方式将opsplash下载到termux home文件夹中
记得使用chmod提权
(3)安装文件管理器
推荐mt管理器，后面的操作也是使用mt管理器操作
2.修改splash
(1)测试opsplash可用
输入 ./opsplash 回车，输出应当有opsplash的用法


(2)把splash分区镜像放置到termux，home文件夹
前往mt管理器，给予root权限
点击下上边的/storage/emulated/0/
在弹窗中输入/data/data/com.termux/files/home/
点击确定
在右半边找到你提取的splash分区，长摁选择复制，复制splash
(3)用opsplash解包splash分区镜像
输入 ./opsplash_aarch64 unpack -i 你的splash镜像 回车
(4)处理图片
用mt管理器查看，发现刚刚的文件夹里出现了许多文件，点击pic文件夹，点击boot.bmp
查看照片尺寸

用工具将你准备好的照片裁剪/缩小至刚刚的分辨率
用mt管理器将改好尺寸的图片复制到/data/data/com.termux/files/home/内，并重命名为input(文件后缀不要修改!!!!)
回到termux，输入 convert 你刚刚裁好尺寸的图片 -type truecolor -depth 24 output.bmp 回车
输入 rm -rf ./pic/boot.bmp 回车
输入 mv ./output.bmp ./pic/boot.bmp
打包splash
输入 ./opsplash repack -i splash.img
3.刷入修改后的splash
(1)复制出splash
去mt管理器，将new-splash.img复制出来
(2)刷入splash
用工具/fastboot刷入new-splash.img
4.重启看效果
